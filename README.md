Sobre o Projeto <br><br>

Este projeto consiste em um sistema web desenvolvido em PHP, que simula a etapa de checkout de uma loja virtual e a geração de uma nota fiscal com cálculo detalhado de frete.<br>

O sistema foi criado com o objetivo de praticar:<br><br>

Envio de dados via formulário (POST)<br>
Estrutura de decisão (switch case)<br>
Organização de lógica de cálculo<br>
